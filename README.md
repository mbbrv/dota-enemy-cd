# Dota Enemy CD

Ручной трекер кулдаунов для второго экрана. Он не читает память Dota 2, пакеты, логи, пиксели экрана или game state; таймеры запускает сам игрок.

## Desktop

Продовый путь - desktop-приложение на Electron:

```powershell
npm install
npm run desktop
```

Глобальные хоткеи в desktop-версии:

- `Ctrl+Alt+клавиша` запускает таймер с такой клавишей.
- `Ctrl+Alt+Shift+клавиша` копирует короткий текст для чата.
- Индикатор `Desktop` в верхней панели означает, что хоткеи идут через Electron.

Desktop-версия регистрирует хоткеи через системный API Electron и пишет в clipboard через main process. Она не читает Dota 2 и не взаимодействует с процессом игры.

## CI Build

При пуше в `main` GitHub Actions собирает portable Windows `.exe`.

Где забрать сборку:

1. Открой репозиторий на GitHub.
2. Перейди в `Actions`.
3. Открой последний run `Build Windows EXE`.
4. Скачай artifact `DotaEnemyCD-windows-portable`.

Локально тот же билд запускается командой:

```powershell
npm run pack:win
```

## Browser Fallback

Можно открыть `index.html` в браузере и перенести окно на второй монитор. В этом режиме обычные хоткеи работают только пока браузер в фокусе.

Legacy-бридж `legacy/hotkeys.ps1` оставлен как временный fallback для браузерного режима, но основной путь - Electron.

## Заметки

- В режиме `Настройка` выбери пресеты героев, поправь кд и назначь клавиши.
- База пресетов лежит в `presets.js`: герои отдельно, предметы/ресурсы отдельно в `addons`.
- Buyback добавляется к конкретному герою через `+ item/resource`, кулдаун 480 секунд.
- В режиме `Матч` нажми `Начать матч`, чтобы включить игровые часы.
- Активный таймер показывает, когда скилл будет готов по игровому времени матча.
- Кнопка copy копирует короткий английский текст для чата, например `Black Hole cd 1:42 ready 12:34`.
- В базу намеренно не добавлены Roshan/Aegis и Blink Dagger.
- Стартовые кулдауны - редактируемые заготовки; сверяй значения с текущим патчем Dota.
