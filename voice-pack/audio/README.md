# Voice Pack Audio

Put local voice files here before building a bundled voice pack.

Recommended names:

- `black-hole-ready.mp3`
- `chronosphere-ready.mp3`
- `bkb-ready.mp3`
- `buyback-ready.mp3`
- `cooldown-ready.mp3`

Supported formats: `.mp3`, `.wav`, `.ogg`, `.m4a`.

In the desktop app, custom user files can also be placed in the app's voice-pack folder with the same names. Use the folder button near the alert selector to open it.
